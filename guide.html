Complete Guide

Everything you need to know about QR codes - how they work, how to use them, and best practices.

1 How QR Codes Work

QR (Quick Response) codes are two-dimensional barcodes invented in 1994 by Denso Wave, a Japanese company. Unlike traditional barcodes that can only store data horizontally, QR codes store information both horizontally and vertically, allowing them to hold significantly more data.

Data Encoding

Information is converted to binary and arranged in specific patterns

Error Correction

Reed-Solomon codes allow recovery even if up to 30% is damaged

Fast Scanning

Can be read instantly from any angle using position patterns

QR Code Anatomy

  • Position Patterns: Three large squares at corners for orientation detection
  • Alignment Pattern: Smaller square(s) for size and distortion correction
  • Timing Pattern: Alternating modules for coordinate calibration
  • Format Information: Error correction level and mask pattern
  • Data Area: The actual encoded information

2 How to Scan QR Codes

Using Phone Camera

  1. Open your phone's default camera app
  2. Point the camera at the QR code
  3. Hold steady until it detects the code
  4. Tap the notification/link that appears
  5. The content will open automatically

Works on iPhone (iOS 11+) and Android (8.0+)

Using WhatsApp

  1. Open WhatsApp
  2. Tap the three dots menu (Android) or Settings (iOS)
  3. Select "Linked Devices" or "WhatsApp Web"
  4. Tap the QR code scanner icon
  5. Point at the QR code

For WhatsApp chat QR codes only

From Gallery/Screenshot

  1. Take a screenshot of the QR code
  2. Use our scanner on this website
  3. Click "Upload Image" button
  4. Select the image from gallery
  5. View the decoded result

Works with any QR code image

Using Google Lens

  1. Open Google app or Google Lens
  2. Point camera at the QR code
  3. Or select image from gallery
  4. Lens will detect and decode
  5. Follow the displayed action

Available on Android and iOS

3 QR Code Types Explained

URL QR Code

Encodes website links. When scanned, opens the URL in a browser.

https://example.com

WhatsApp QR Code

Opens WhatsApp chat with specified number and optional message.

https://wa.me/919876543210?text=Hello

UPI Payment QR

Enables instant UPI payments through apps like GPay, PhonePe, Paytm.

upi://pay?pa=name@upi&pn=Name&am=100&cu=INR

WiFi QR Code

Auto-connects to WiFi network when scanned.

WIFI:T:WPA;S:NetworkName;P:Password;;

Location QR Code

Opens location in maps application.

geo:28.6139,77.2090

vCard QR Code

Saves contact information directly to phone.

BEGIN:VCARD...END:VCARD

4 Error Correction Levels

QR codes use Reed-Solomon error correction to remain readable even when partially damaged or obscured.

Level Recovery Best For Data Capacity
L (Low) ~7% Clean environment, maximum data Highest
M (Medium) ~15% General purpose (Recommended) High
Q (Quartile) ~25% Industrial, outdoor use Medium
H (High) ~30% Logo overlay, harsh conditions Lowest

5 Best Practices

Size Matters

Minimum 2cm x 2cm for printed codes. Larger for distance scanning.

High Contrast

Use dark foreground on light background for best readability.

Quiet Zone

Keep white margin around QR code (at least 4 modules wide).

Test Before Use

Always scan your QR code with multiple devices before printing.

Short URLs

Use URL shorteners for long links to create simpler QR codes.

Print Quality

Use high resolution (300 DPI+) for printed QR codes.
